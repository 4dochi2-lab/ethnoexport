# EthnoExport

Kənd sənətkarlarını dünya bazarına birləşdirən platforma. Sənətkar şəkil çəkir və istədiyi məbləği yazır; qalanını — AI elan mətni, tərs qiymət hesablaması, logistika və satış — EthnoExport komandası edir.

## Səhifələr
- `index.html` — marketinq/tanıtım səhifəsi (canlı qiymət kalkulyatoru, işlək demo, admin dashboard maketi)
- `daxil.html` — sənətkar giriş / qeydiyyat
- `panel.html` — sənətkar paneli (məhsul göndər, statusu izlə)
- `admin.html` — komanda paneli (QC, Etsy-də dərc, satıldı/ödənildi)

## Qeyd
Bu prototipdir — hesab və məhsul məlumatları brauzerin `localStorage`-ında saxlanılır (hələ backend yoxdur).

**Admin girişi:** `admin@ethnoexport.az` / `admin123`

Statik saytdır — Vercel-ə birbaşa (config-siz) deploy olunur.
